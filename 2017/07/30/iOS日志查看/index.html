发表于 2017-07-30

查看 iOS 设备System Log方法

  1. syslogd to /var/log/syslog (Cydia安装，已不适用)
  2. idevicesyslog (github开源，有些问题)
  3. Xcode -> Deveices -> Deice -> 左下角三角形按钮
  4. socat (越狱设备上查看)
